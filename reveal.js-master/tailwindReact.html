Setup Tailwind css in react (VITEJS) 

						A Guide to Installing Tailwind CSS with React
Tailwind CSS is a popular utility-first CSS framework 

						Using utility classes to build custom designs 
						
						without writing CSS
Why Choose Tailwind CSS? 

						- Speed: Create components quickly

						- Customization: Easy custom

						- Scalability: Ideal for All Project Sizes
- Speed: Quickly build stylish UI components without the need for extensive CSS files. - Customization: Tailwind is highly customizable, allowing you to tailor your styles to your project’s needs. - Scalability: Ideal for projects of all sizes, from small websites to large applications.
Step 1: Create a React Application 


						- npx create-vite your-project-name --template react

						(npx) = Node Package eXecute.

						execute Node.js packages without needing to install them globally

						and and the template you want to use here (react)
Step 2: Install Tailwind CSS 

					- npm install -D tailwindcss postcss autoprefixer

					- Install tailwindcss and its peer dependencies,
Step 3: Configure Tailwind CSS 

					- npx tailwindcss init -p

					- Then generate your tailwind.config.js files.

					/** @type {import('tailwindcss').Config} */
					export default {
						content: [
						],
						theme: {
							extend: {},
						},
						plugins: [],
					}
Step 4: Configure your template paths 

					- Add the paths to all of your template files 
					in your tailwind.config.js file.

					/** @type {import('tailwindcss').Config} */
					export default {
						content: [
						// new 
							"./index.html",
							"./src/**/*.{js,ts,jsx,tsx}",
						],
						theme: {
							extend: {},
						},
						plugins: [],
					}
Step 5: Including Tailwind in Your CSS 

					-	Add the @tailwind directives for each of Tailwind’s layers 
					to your ./src/index.css file.

					/* src/index.css */
	
					@tailwind base;
					@tailwind components;
					@tailwind utilities;
Step 6 Start your build process 

						- using Tailwind in your project 
						
						- and change in app

						export default function App() {
							return (
								

									Hello world!
								

							)
						}

						- and npm run dev
Step 7 Tailwind CSS is an ecosystem : 

						- Install the Official Plugins

						- Plugins can be added to your project via npm, 
						then adding them to your tailwind.config.js file:

						/** @type {import('tailwindcss').Config} */
						module.exports = {
							// ...
							plugins: [
								require('@tailwindcss/typography'),
								require('@tailwindcss/forms'),
							]
						}
Step 8 Typography plugin 

						- npm install -D @tailwindcss/typography

						- https://github.com/tailwindlabs/tailwindcss-typography?tab=readme-ov-file

						- https://tailwindcss-typography.vercel.app/
The official Tailwind CSS Typography plugin provides a set of prose classes you can use to add beautiful typographic defaults to any vanilla HTML you don’t control, like HTML rendered from Markdown, or pulled from a CMS
Step 9 Forms plugins 

					
					- npm install -D @tailwindcss/forms

					- https://github.com/tailwindlabs/tailwindcss-forms

					- https://tailwindcss-forms.vercel.app/
A plugin that provides a basic reset for form styles that makes form elements easy to override with utilities.
Step 10 Reusing Styles 

						- Managing duplication and creating reusable abstractions.
						- 
						
						 
						
						
						@layer components {
							.btn-primary {
								@apply py-2 px-5 bg-violet-500 text-white font-semibold rounded-full shadow-md hover:bg-violet-700 focus:outline-none focus:ring focus:ring-violet-400 focus:ring-opacity-75;
							}
						}
