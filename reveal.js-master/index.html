Partials in Sass 

						- start the file name with an underscore _
						
						- use the @import directive

						- import "_nameFile.scss";
- For code organization You'll notice that the underscore and the .scss are not added. That is because Sass automatically assumes that you are referring to the .sass or .scss file.
Structure your Sass Projects (Architecture) 

					sass/
					|- base/
					|    |- _base.scss
					|    |- _reset.scss
					|- components/
					|    |- _buttons.scss
					|- layout/
					|    |- _header.scss
					|- vendors/
					|    |- _font.scss
					|    |- _fontawesome.scss
					|- style.scss
					|
- To a structure files in a project.
File style.scss 

						/*! SASS #style main */
